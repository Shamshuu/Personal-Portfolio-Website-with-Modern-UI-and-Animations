# Shamshudeen Shaik - Personal Portfolio Website

A fully responsive, animated personal portfolio website built from scratch to showcase my skills, projects, and contact information in a modern, professional design.

## Tech Stack

- React + Vite
- CSS3 (custom responsive styling + glassmorphism)
- Framer Motion (scroll animations + parallax)
- React Icons

## Live Website

Deployment URL: `https://luxury-pithivier-441bad.netlify.app/`

Repository: `https://github.com/Shamshuu/Personal-Portfolio-Website-with-Modern-UI-and-Animations`

## Features

- Responsive design across mobile, tablet, and desktop breakpoints
- Sticky navigation with mobile menu support
- Hero, About, Skills, Projects, Contact, and Footer sections
- At least one parallax scrolling effect
- Multiple on-scroll reveal animations
- Reduced motion accessibility support with `prefers-reduced-motion`

## Local Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/Shamshuu/Personal-Portfolio-Website-with-Modern-UI-and-Animations.git
   ```
2. Move into the project directory:
   ```bash
   cd Personal-Portfolio-Website-with-Modern-UI-and-Animations
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start development server:
   ```bash
   npm run dev
   ```
5. Build for production:
   ```bash
   npm run build
   ```

## Netlify Deployment

1. Push this project to the `main` branch of GitHub.
2. In Netlify, select **Add new site -> Import an existing project**.
3. Choose the repository:
   `Shamshuu/Personal-Portfolio-Website-with-Modern-UI-and-Animations`.
4. Build settings:
   - Build command: `npm run build`
   - Publish directory: `dist`
5. Click **Deploy site**.

The project includes `netlify.toml` with the required build and SPA redirect configuration.
